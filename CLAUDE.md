# AGROARM Project Guidelines

## Color Palette

All UI work must use the official AGROARM palette. Do not use arbitrary colors or hardcoded hex values outside this set.

| Name | Hex | Usage |
|---|---|---|
| Primary Green | `#007D41` | Buttons, links, hover states, accents |
| Dark Green | `#024E29` | Header, footer, dark sections |
| Light Background | `#F4F8F6` | Content section backgrounds (use instead of pure white) |
| Pure White | `#FFFFFF` | Cards, product tiles, form fields |
| Neutral Light Gray | `#E6EFEA` | Borders, dividers, subtle background blocks |
| Dark Text | `#1F2A24` | Body text (not black — softer and more professional) |
| Hover Green | `#009F54` | Hover state on buttons and interactive elements |

### Rules
- Never use `#000000` for text — use `#1F2A24` instead.
- Never use `#FFFFFF` for page/section backgrounds — use `#F4F8F6` instead. Reserve white for cards and form fields.
- All green accents must use `#007D41` (primary) or `#024E29` (dark). Do not invent new greens.
- Hover states on green elements should use `#009F54`.

## Language

All user-facing content on the site must be in Serbian (Latin script).
